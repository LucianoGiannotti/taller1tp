Usage:
  main -h
  main -V
  main [options] filename_XML
Options:
  -h, --help   		Prints usage information.
  -V, --version		Prints version information.
  -l, --log_level	Log level to use in the logger.
  -c, --config_xml	Path to the config.xml .

Examples:
./main -c config_xml -l log_level
./main -c config_xml
./main -l log_level
./main
